2.1 rcvrOpenAcquire

描述

接收处理卡初始化函数。在本函数中预留数据缓冲区。初始化WD库，打开内核模式驱动，验证用户是否合法。初始化用户及内核中采集相关的数据结构。只有在本函数执行成功后，其它函数才能有效执行。

语法

DWORD rcvrOpenAcquire (MEM_SIZE memSize); // memSize是预留数据缓冲区

CPCIeDevice::CPCIeDevice()

参数

memSize – MEM_32Mega, 32 M字节

MEM_64Mega, 64 M字节

MEM_128Mega, 128 M字节

MEM_256Mega, 256 M字节

MEM_512Mega, 512 M字节

返回状态码

RCVR_LIBINIT_ERR （库错误）

RCVR_DEVICE_NOT_OPEN （设备没有打开）

RCVR_RESERVE_MEMORY_ERR （内存错误）

RCVR_BUNDLE_ARRAY_ERR

RCVR_KERNEL_ERR （内核错误）

RCVR_SUCCESS （成功）

2.2 rcvrCloseAcquire

描述

接收处理卡清除关闭函数。此函数与rcvrOpenAcquire函数必须成对出现，如果不调用此函数，会造成采集卡再次打开时失败。

语法

Microsoft C/C++

DWORD rcvrCloseAcquire (void);

CPCIeDevice::~CPCIeDevice()

参数

无

返回状态码

RCVR_DEVICE_NOT_OPEN （设备没有打开）

RCVR_SUCCESS （关闭成功）

参数

cmd ：用户控制采集的命令。

DACQ_START 启动采集

DACQ_STOP 停止采集

bTrigger ：是否采用触发模式GPS

返回状态码

RCVR_DEVICE_NOT_OPEN

RCVR_CONTROL_CMD_ERR

RCVR_CONTROL_CMD_INVALID

RCVR_SUCCESS

2.4 rcvrAquireReadableMemorySegment

描述

获取可读取的内存数据段，当数据缓冲区中有可用的采集数据时，该函数返回内存数据的地址和长度，用户可以指定是否等待的超时事件。

语法

DWORD rcvrAquireReadableMemorySegment (PVOID* ppUserAddr, DWORD* pdwLength, DWORD dwMSWait);

参数

ppUserAddr：返回数据的内存地址。

pdwLength: 返回数据的长度。

dwMSWait: 用户等待数据的时间，单位为ms

0xFFFFFFFF表示一直等待

返回状态码

RCVR_DEVICE_NOT_OPEN

RCVR_GENERAL_ERR

RCVR_BE_NORMAL

RCVR_WRITE_FULL

RCVR_BE_STOPPED

RCVR_READBLE_TIMEOUT

RCVR_SUCCESS

Microsoft C/C++

DWORD rcvrUpdateCurrentMemorySegment (void);

NULL
